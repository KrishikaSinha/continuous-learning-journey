JavaScript Notes

1. Introduction to JavaScript

JavaScript is a high-level, dynamic programming language used for:

  • Making webpages interactive
  • Handling events
  • Creating dynamic content
  • Backend development (Node.js)

3. Data Types

JavaScript has 2 types of data:

Primitive

  • String
  • Number
  • Boolean
  • Null
  • Undefined
  • Symbol
  • BigInt

Non-Primitive

  • Objects
  • Arrays
  • Functions
let name = "Krishika"; let age = 18; let isStudent = true;

4. Operators

  • Arithmetic (+, -, *, /, %)
  • Comparison (==, ===, !=, >, <)
  • Logical (&&, ||, !)
  • Assignment (=, +=, -=)
let a = 10; let b = 5; console.log(a + b); // 15

5. Functions

Normal Function

function greet() { console.log("Hello World"); } greet();

Arrow Function

const add = (a, b) => a + b; console.log(add(5, 10));

6. DOM Manipulation

Selecting Elements

document.getElementById("title"); document.querySelector(".box");

Changing Content

document.getElementById("demo").innerHTML = "Updated!";

7. Events

Events are actions performed by the user.

document.getElementById("btn").onclick = function() { alert("Button Clicked"); }

8. Loops

For Loop

for(let i = 1; i <= 5; i++) { console.log(i); }

While Loop

let x = 1; while(x <= 5) { console.log(x); x++; }

9. Arrays

let fruits = ["Apple", "Banana", "Mango"]; fruits.push("Orange"); console.log(fruits);

10. Objects

let person = { name: "Krishika", age: 18, city: "Gurgaon" }; console.log(person.name);

11. Conditionals

let age = 20; if(age >= 18) { console.log("Adult"); } else { console.log("Minor"); }

  • Write a program to check even or odd.
  • Create an object student with 3 properties.
