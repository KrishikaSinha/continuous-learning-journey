CSS Notes

1. What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, responsiveness, and animations.

2. CSS Basic Syntax

selector { property: value; }

3. Types of CSS

  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS Example:

<h1 style="color:red;">Hello</h1>

Internal CSS Example:

<style> p { color: blue; } </style>

External CSS Example:

<link rel="stylesheet" href="style.css">

4. Colors in CSS

  • Color names (red, blue)
  • Hex codes (#ff0000)
  • RGB (rgb(255,0,0))
  • RGBA (rgba(255,0,0,0.5))
p { color: #ff5733; }

5. Font Properties

  • font-size
  • font-weight
  • font-style
  • font-family
p { font-size: 20px; font-weight: bold; }

6. Box Model

Every element has:

  • Content
  • Padding
  • Border
  • Margin
div { padding: 20px; margin: 10px; border: 2px solid black; }

7. Display Property

  • block
  • inline
  • inline-block
  • flex
  • grid
div { display: flex; }

8. Positioning

  • static
  • relative
  • absolute
  • fixed
  • sticky
.box { position: absolute; top: 20px; left: 40px; }

9. Flexbox

  • justify-content
  • align-items
  • flex-direction
  • gap
.container { display: flex; justify-content: center; align-items: center; }

10. CSS Grid

.grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 12px; }

11. Animations

@keyframes fade { from { opacity: 0; } to { opacity: 1; } } div { animation: fade 2s ease-in; }

12. Practice Questions

  • Create a box with padding, margin, and border.
  • Use flexbox to center a div.
  • Make a 3-column grid layout.
