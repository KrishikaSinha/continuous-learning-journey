HTML Notes

1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to structure webpages. It defines the content and layout of a website.

2. Basic HTML Structure

Every HTML document follows this structure:

<!DOCTYPE html> <html>   <head>     <title>My Page</title>   </head>   <body>     Content goes here   </body> </html>

3. HTML Tags

HTML uses tags to define elements. Example:

  • <h1> – Heading
  • <p> – Paragraph
  • <a> – Link
  • <img> – Image
  • <div> – Container

4. Headings

HTML provides 6 levels of headings:

<h1> Main Heading </h1> <h2> Sub Heading </h2> <h3> Section Title </h3>

5. Paragraphs

<p> This is a paragraph. </p>

6. Links

<a href="https://google.com"> Visit Google </a>

7. Images

<img src="image.jpg" width="300" />

8. Lists in HTML

Ordered List:

<ol>   <li>Item 1</li>   <li>Item 2</li> </ol>

Unordered List:

<ul>   <li>Item 1</li>   <li>Item 2</li> </ul>

9. Table

<table border="1">   <tr>     <th>Name</th>     <th>Age</th>   </tr>   <tr>     <td>John</td>     <td>20</td>   </tr> </table>

10. Forms

<form>   Name: <input type="text"> <br>   Email: <input type="email"> <br>   <button>Submit</button> </form>

11. Semantic Tags

  • <header>

  • <nav>
  • <section>
  • <article>
